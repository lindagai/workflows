Contains:

(1) cleaning trio data workflow

(2) trio analysis workflow, including rare variants analysis

(3) JHPCE quick-start guide

(4) directions for future work (.txt file)
(e.g. visualization packages, other rare variants methods, etc.)

=====
 
The .Rmds for each workflow use relative paths and should run on other computers, as long as you:

(1) set the working directory to the location of the .Rmd

(2) have RV-TDT downloaded on your computer and provide the file path in the .Rmd where needed. More detailed READMEs are available in each folder.
